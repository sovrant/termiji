# Termiji
## Introduction
`Termiji` is a fast, easy-to-use terminal emoji picker with real-time search and scalable UI, written is Rust. It lets user quickly launch the program and just choose their desired emoji. Designed for those who prefer speed and simplicity over bloated UI applications, `Termiji` helps you gets things done efficiently.

## Installation
### Dependencies
#### Linux
- Rust and cargo as the build dependencies
    ```bash
    git clone https://github.com/sovrant/termiji.git
    cd termiji
    cargo build --release
    ./target/release/termiji
    ```
